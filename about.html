About CivilCalc Pro

From Theory to Tool - A Student-Led Engineering Initiative

Developer Profile

Who is Behind This?

I am a first-semester student of BS Artificial Intelligence at COMSATS University Islamabad (Wah Campus). Being an AI student, I have a deep interest in how logic and algorithms can simplify complex real-world problems.

This project, CivilCalc Pro, is my attempt to bridge the gap between traditional civil engineering calculations and modern web technology. While my core studies are in AI, I believe that building utilities for various industries (like Civil Engineering) is the best way to practice pure programming logic.

The Project Mission

Many site engineers and students still rely on manual formulas or heavy software for basic daily calculations. CivilCalc Pro is designed to be an all-in-one utility tool that is:

  • Fast: Built with Vanilla JavaScript for instant results.
  • Accessible: Works on any mobile or desktop browser.
  • Precise: Uses standard engineering formulas for material and structural estimation.

Currently, the site is under active development, with a goal to host 50+ specialized tools covering everything from concrete mix design to structural steel analysis.

Technology & Logic

This entire platform is built using HTML5, CSS3, and Pure JavaScript. No heavy frameworks were used to ensure that the site loads quickly even on slow construction site internet connections.

// Sample Logic: Bending Moment for Simply Supported Beam
function calculateBM(weight, length) {
  return (weight * Math.pow(length, 2)) / 8;
}
